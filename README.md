# Example of Elixir

## 1. Towers of Hanoi (problem with recursion)

- Source file
  - hanoi.exs


- Execution result
```
$ elixir hanoi.exs
moving disk from A to C
moving disk from A to B
moving disk from C to B
moving disk from A to C
moving disk from B to A
moving disk from B to C
moving disk from A to C
```
